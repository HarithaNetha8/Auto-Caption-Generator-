🎙️ Auto Caption Generator
--
📌 Project Description
--
Auto Caption Generator is a Python-based application that automatically converts speech from audio files into text captions. The generated captions are saved in standard SubRip (.srt) format, making them suitable for videos, lectures, podcasts, and accessibility purposes.

This project demonstrates basic audio processing and speech-to-text conversion using Python.

📁 Project Structure
--
Auto-Caption-Generator/
├── auto_caption_generator.py   # Main Python script

├── test.mp3                    # Sample audio input

├── test2.mp3                   # Additional test audio

├── captions.srt                # Generated subtitle file

└── README.md

✨ Features
--
Converts audio files (.mp3) into captions

Generates .srt subtitle files

Simple and easy-to-run Python script

Suitable for video captioning and transcription

Helps improve accessibility

🧰 Technologies Used
--
Python 3

Speech Recognition / Audio Processing Libraries

SubRip Subtitle (.srt) format

▶️ How to Run the Project
--
📌 Prerequisites

Python 3.8 or higher

Required Python libraries installed (as used in the script)

🟢 Execution Steps
git clone (https://github.com/HarithaNetha8/Auto-Caption-Generator)
cd auto-caption-generator
python auto_caption_generator.py


Ensure the audio file path is correctly set inside auto_caption_generator.py.

🧪 Sample Input & Output

Input: test.mp3, test2.mp3

Output: captions.srt

The generated .srt file can be used directly in video players or video editing tools.

📌 Use Cases

Caption generation for videos

Accessibility support for hearing-impaired users
